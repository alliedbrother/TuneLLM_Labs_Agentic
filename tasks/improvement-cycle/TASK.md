Weekly improvement cycle review. The CEO should:

1. Review the latest evaluation report and weakness analysis
2. Decide which weakness to target in the next improvement cycle
3. Kick off data curation/creation tasks
4. Review any in-progress training runs
5. Review completed evaluations and make promotion decisions
6. Check cost reports and infrastructure health
