The primary project of this company. Each improvement cycle targets a specific model weakness identified by the Evaluation Agent, flows through data curation/creation, fine-tuning, and validation, and produces a new model version that is either promoted to production or rejected.

## Success Criteria

- Each cycle produces a measurable improvement on the targeted weakness
- No regressions on any core benchmark exceeding 2%
- All safety evaluations pass
- Full traceability from weakness → data → training → eval → deployment
